Secciones del proyecto

Inicio: Página principal del sitio.

Paginas: Se pueden ver todas las páginas creadas. Cada una tiene un botón de Leer más para ver el detalle. Si no hay paginas se muestra "No hay páginas aún".

Login: Permite a los usuarios existenes acceder a su cuenta.

Signup: Permite crear nuevos usuarios.

Perfil: Muestra los datos del usuario y su biografía.

Editar perfil: Permite cambiar username, email o bio.

Cambiar contraseña: Permite actualizar la contraseña del usuario.

Admin: Panel de administración de Django para gestionar usuarios y contenido.
